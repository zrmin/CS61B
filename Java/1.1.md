1. `javac`是java的compiler

2. `java`是java的解释器

   * 在命令行中运行java代码时，先使用`javac`编译器将java编译成字节码文件
   * 然后用`java`解释器解释字节码文件运行java程序

3. `System.out.println(5+"10")`输出`510`，是字符串

4. `System.out.println(5 + 10)`输出`15`， 是数字
